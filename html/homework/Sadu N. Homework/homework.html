Мое первое домашнее задание

CSS первоначально расшифровывался как “Cascading Style Sheets” (Каскадные таблицы стилей) и был создан Хоконом Вием Ли (Håkon Wium Lie) в 1994 году. Цель состояла в том, чтобы отделить структуру HTML-документа от его внешнего вида, позволяя разработчикам легко изменять оформление веб-страниц без необходимости менять сам HTML-код.

  1. HTML — это язык гипертекстовой разметки для создания веб-страниц.
  2. HTML был создан Тимом Бернерс-Ли в 1991 году.
  3. HTML5 — актуальная версия языка, представлена в 2014 году.
  4. HTML — это не язык программирования, а язык разметки.
  5. HTML-код работает в любом браузере и на любой операционной системе.
